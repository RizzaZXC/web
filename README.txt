1. устанавливаем Python 3.12.2
2. переходим в папку с проектом

Работа с помощью командной строки:
3. открываем командную строку и прописываем следующее: 
venv\Scripts\activate
нам открывается виртуальное окружение
4. запускаем приложение с помощью команды
python main.py
5. чтобы выйти из виртуального окружения, прописываем 
deactivate

Работа в PyCharm
3. Открываем main.py с помощью PyCharm
4. В появившемся окне выбираем "Open in Project"
5. В следующем окне выбираем путь к проекту
6. Нажимаем кнопку "Trust project"
7. Ждем, когда проект соберется. Справа снизу появится intepreter "Python 3.12 (venv)". 
Папка "flask_project" подсветится желтым.
8. Если проект все-таки не собрался, нажимаем на <No intepreter>.
В появившемся окне выбираем add local intepreter -> existing -> add local intepreter и выбираем интерпертатор из папки flask_project
"<ваш путь к проекту>\flask_project\venv\Scripts\python.exe"

Работа в VS Code
3. Нажимаем на правую кнопку мыши в папке проекта и выбираем "Открыть с помощью Code"
4. Справа снизу необходимо настроить интерпретатор. Нажимаем на написанную версию и в появившемся сверху окне выбираем "Python 3.12.2 ('venv': venv)"
Если интерпретатор из виртуального окружения не нашелся, добавляем его вручную.
